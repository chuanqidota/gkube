🔑 核心交互流程

用户操作流程
🔐 登录流程
输入用户名密码 → POST /v1/auth/login → 获取 JWT Token → 存储到 localStorage → 跳转 Dashboard
🖥️ 集群操作流程
集群列表 → 点击"添加集群" → 填写名称+粘贴kubeconfig → 验证连通性 → 保存
集群列表 → 点击某个集群 → 进入集群详情 → 查看节点/命名空间/资源概览
📦 资源浏览流程
顶部选择集群 → 侧边栏选择资源类型 → 查看列表 → 点击详情 → 查看YAML/日志/终端
