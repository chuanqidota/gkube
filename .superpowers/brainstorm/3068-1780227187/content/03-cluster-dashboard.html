🖥️ 集群管理流程

集群注册与健康检测
📝 集群注册流程
用户填写信息 → 粘贴kubeconfig → 验证连通性 → AES加密存储 → 首次健康采集 → 注册完成
💓 健康检测机制
定时探测
  • • 每 30 秒探测一次
  • • 调用 /healthz 和 /version
  • • 通过 goroutine 并发执行
  • • 失败 3 次标记 offline
数据采集
  • • 节点数量与状态
  • • CPU/内存容量与使用率
  • • Pod 数量与运行状态
  • • K8s 版本信息
状态缓存
  • • 健康数据缓存到 Redis
  • • TTL: 60 秒自动过期
  • • Dashboard 读缓存数据
  • • 避免频繁请求集群 API
🔗 连通性检测 API
POST /v1/clusters/:id/check

// 手动触发连通性检测，返回结果：
