🔗 API 路由设计（重新规划）

模块 路由前缀 说明
认证 /v1/auth/* 登录、注册、Token刷新
用户管理 /v1/users/* 用户CRUD、角色分配
角色管理 /v1/roles/* 角色CRUD、权限配置
集群管理 /v1/clusters/* 集群注册/编辑/删除/健康检测
Dashboard /v1/dashboard/* 全局概览、资源统计、事件聚合
K8s资源 /v1/k8s/:cluster/* Pod/Deployment/Service等CRUD
终端 /v1/k8s/:cluster/exec/* WebSocket终端、日志流
